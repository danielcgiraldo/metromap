`https://embed.metromap.online/v1/map?public-key=<public-key>`

### Parámetros de la URL

| Nombre    | Tipo   | Descripción                                                                                               |
| --------- | ------ | --------------------------------------------------------------------------------------------------------- |
| `public-key` {<><br/><code className="opt docs">OPCIONAL</code></>}    | string | Para autenticar el mapa y permitir su inserción en un dominio específico, se requiere una clave pública válida. Además, es necesario haber configurado el dominio permitido previamente para poder insertar el mapa en él."                                                      |

### Código de Ejemplo

```html
<iframe style="aspect-ratio: 3/4; width: 100%; border: 0;"src="https://embed.metromap.online/v1/map?public-key=<public-key>" />
```
