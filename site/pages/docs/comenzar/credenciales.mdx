# Credenciales
## ¿Qué son las credenciales?
Una credencial de API es una clave secreta que se utiliza para autenticar y autorizar el acceso a los recursos y datos de una API.
Las credenciales de API se utilizan para garantizar que solo los usuarios autorizados puedan acceder y utilizar los servicios y recursos de una API determinada. De esta manera, se asegura la privacidad y la seguridad de los datos y se evita cualquier intento de acceso no autorizado a la API.

## Obtén tus credenciales
Como podrás intuir, las credenciales son una parte fundamental para hacer uso de nuestras API. Para obtener tus credenciales, solo debes seguir los siguientes pasos:

### Paso 1: Iniciar Sesión
Para comenzar el proceso, lo primero que debes hacer es iniciar sesión con tu usuario y contraseña o con tu cuenta de GitHub. Para hacerlo, simplemente haz clic en el botón a continuación. Si aún no te has registrado, también puedes utilizar este mismo botón para crear una cuenta nueva.

### Paso 3: Guarda tus Keys
Después de recibir tus claves para la API, es importante que las guardes para evitar que se pierdan o sean compartidas con personas o grupos no autorizados.




### Paso 4: Llama a la API
Finalmente, solo necesitas utilizar las claves que te proporcionamos y llamar a la API elegida desde tu aplicación.
