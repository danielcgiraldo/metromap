# Comenzar Ahora

Las APIs de MetroMap son una herramienta útil para obtener información precisa y actualizada sobre el Metro de Medellín. A través de ellas, los usuarios pueden consultar no solo el estado actual de las líneas y estaciones, sino también los incidentes que puedan afectar el servicio y los cambios en los estados de las estaciones y líneas en tiempo real.

Con estas APIs, los desarrolladores pueden integrar fácilmente la información en sus propias aplicaciones, lo que permite una planificación de viajes más efectiva y precisa.

Para obtener información mas detallada sobre cada API en especifico puede consultar la sección de [API](api).
